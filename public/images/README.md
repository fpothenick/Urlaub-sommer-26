# Bilder hier platzieren

In diesem Ordner müssen folgende Bilder platziert werden:

## Benötigte Bilder:

- `group.jpg` - Gruppenbild für die Willkommensseite
- `p1_1.jpg` - Paket 1, Unterkunft 1
- `p1_2.jpg` - Paket 1, Unterkunft 2
- `p2_1.jpg` - Paket 2, Unterkunft 1
- `p2_2.jpg` - Paket 2, Unterkunft 2
- `p3_1.jpg` - Paket 3, Unterkunft 1
- `p3_2.jpg` - Paket 3, Unterkunft 2

## Empfehlungen:

- Format: JPG oder PNG
- Bildverhältnis: 4:3 oder 16:9
- Minimale Größe: 800x600px
- Die App funktioniert auch ohne Bilder (zeigt Platzhalter)

## Nach dem Hinzufügen:

Einfach die Bilder in diesen Ordner kopieren. Der Dev-Server muss nicht neu gestartet werden.
